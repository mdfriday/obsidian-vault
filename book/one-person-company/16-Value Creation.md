## 🧠 一文看懂《技能价值创造框架》

### 一、核心理念：

> “做你喜欢的事，但要懂心理学、营销和销售，否则只能成为穷困的艺术家。”

关键思想：

* 技能只有与**人类心理与普遍规律**相连，才能创造价值；
* 真正赚钱的技能是那些能**影响认知与行为**的技能；
* 价值 = 认知到问题 + 路径 + 优先级。

---

### 二、人类行为的三要素（Purpose / Path / Priority）

1. **Purpose（目的）**：来自对问题的觉察与欲望；
2. **Path（路径）**：提供克服问题的结构化方法；
3. **Priority（优先级）**：不同人看重不同问题，决定了你营销的角度。

> ✅ 成功的内容 / 产品 = 清晰的问题 + 明确的路径 + 对重要性的放大。

---

### 三、意识的五个层次（客户认知阶段）

1. **无意识（Unaware）**：不知道有问题；
2. **问题意识（Problem Aware）**：知道问题但没行动；
3. **解决方案意识（Solution Aware）**：开始了解可能的解法；
4. **产品意识（Product Aware）**：知道你的方案；
5. **最清楚（Most Aware）**：需要一点推动（优惠、限时等）。

> 内容营销的目标：针对不同意识层次，输出不同深度的内容。

---

### 四、人类的八大欲望（营销定位的底层逻辑）

1. 生存（Survival）
2. 生命体验（Life / Enjoyment）
3. 免于恐惧（Freedom from fear）
4. 性 / 伴侣（Sexual companionship）
5. 舒适与清醒（Comfort / Clarity）
6. 地位与权力（Perceived status）
7. 社区安全（Community security）
8. 社会认可（Social approval）

> 所有营销、文案、内容的切入点，都应基于这八大欲望。

---

### 五、价值创造的 8 步框架

| 步骤  | 内容   | 作用                        |
| --- | ---- | ------------------------- |
| 1️⃣ | 认知层次 | 明确目标受众处于哪个意识阶段            |
| 2️⃣ | 定位   | 从八大欲望中找到切入点               |
| 3️⃣ | 大问题  | 找出他们最在意、最痛的矛盾             |
| 4️⃣ | 独特机制 | 提供与众不同的解决路径               |
| 5️⃣ | 具体好处 | 用子弹点展示利益（Bullet Benefits） |
| 6️⃣ | 证明   | 用故事、数据、经验或客户见证支撑可信度       |
| 7️⃣ | 伟大想法 | 用一句话总结你的核心理念（Big Idea）    |
| 8️⃣ | 风险逆转 | 提供保证或安全机制，降低购买心理门槛        |

> 💡公式总结：
> **（意识 × 欲望） + 问题 + 独特机制 + 证明 + 保证 = 高感知价值产品。**

---

### 六、总结一句话

> “你的产品不是卖功能，而是卖改变；
> 你的内容不是传知识，而是唤醒意识。”
